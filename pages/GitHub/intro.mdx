# Git とは

Git は、ソースコードのバージョン管理を行うための分散型バージョン管理システムです。複数人での開発や、過去の変更履歴の管理が簡単にできます。

---

# GitHub とは

GitHub は、Git リポジトリをインターネット上で管理・共有できるサービスです。チームでの共同開発やコードレビュー、課題管理などが行えます。

---

# Visual Studio Code からプロジェクトをクローンする手順

1. **GitHub でリポジトリの URL をコピー**

   - 対象リポジトリのページで「Code」ボタンをクリックし、表示された URL をコピーします。

2. **Visual Studio Code を起動**

3. **コマンドパレットを開く**

   - `Cmd + Shift + P`（Mac）または`Ctrl + Shift + P`（Windows）を押します。

4. **「Git: Clone」を選択**

   - 「Git: Clone」と入力し、コマンドを選択します。

5. **コピーした URL を貼り付けて Enter**

6. **保存先フォルダを選択**

7. **「開く」を選択してプロジェクトを開く**
